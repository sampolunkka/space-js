# Space JS

## Introduction

Space JS is a Space Impact clone familiar from the Nokia's hit phone 3310. This project is vanilla JavaScript based and
runs in the browser. It features simple graphics and gameplay reminiscent of the original game.

## How to Play

The objective of the game is to shoot down as many enemy ships as possible while avoiding their attacks. Use the w and s
keys to move your spaceship and the spacebar to shoot.
